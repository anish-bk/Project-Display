Pipeline

Pipeline Stages in Detail

Stage I

Geometric Inference & Point Cloud Generation

The first stage extracts geometric priors from the single input RGB image. This provides the foundation for accurate 3D reconstruction by establishing initial depth and surface information.

1
Monocular Depth Estimation

Deep learning models predict dense depth maps from the single input image, providing relative depth information for each pixel.

2
Surface Normal Prediction

Estimate surface normals to understand local geometry and surface orientation, crucial for accurate reconstruction.

3
Importance Sampling

Generate point clouds using stratified importance sampling, concentrating points in geometrically significant regions.

4
KNN Refinement

Refine point clouds using K-Nearest Neighbors for outlier removal and density optimization.

Stage II

Diffusion-Guided Optimization & Rendering

The second stage converts the point cloud into a Gaussian representation and optimizes it using camera-conditioned score distillation sampling for multi-view consistent reconstruction.

1
Gaussian Initialization

Convert point cloud to 3D Gaussian representation with position, covariance, opacity, and spherical harmonics.

2
Camera-Conditioned SDS

Apply score distillation sampling with camera pose conditioning to enforce geometric consistency across viewpoints.

3
Iterative Optimization

Refine Gaussian parameters through gradient-based optimization guided by diffusion model feedback.

4
Mesh Extraction

Extract final mesh with baked textures using marching cubes or density thresholding.

Stage III

Feedforward 3D Editing & Generation

The third stage provides a complementary feedforward pipeline for rapid 3D asset generation and direct editing using natural language instructions.

1
Instruction Parsing (Qwen2509)

Multimodal vision-language model parses user instructions and generates structured editing prompts.

2
Direct 3D Generation (Hunyuan3D)

Feedforward generator directly produces complete 3D assets from text or image inputs in a single pass.

3
Asset Manipulation

Apply text-guided edits directly to 3D representations without multi-view intermediate steps.

4
Output Generation

Export final edited 3D asset in standard formats (OBJ, GLB) with materials and textures.

Methodology

Development Phases

The project was developed through multiple iterative phases

1

Classical 3D Gaussian Splatting

Implementation and training of vanilla 3DGS on multi-view datasets to understand the representation.

2

Triangle Splatting Investigation

Exploration of alternative triangle-based representations for comparison with Gaussian approaches.

3

DreamFusion Baseline

Implementation of Score Distillation Sampling (SDS) fundamentals using NeRF-based architecture.

4

DreamGaussian Implementation

Adapting SDS to 3D Gaussian Splatting for faster optimization and mesh conversion capabilities.

5

Monocular Depth Integration

Single image 3D reconstruction via monocular depth estimation and surface normal prediction.

6

Importance Sampling & Point Cloud

Development of importance sampling framework and KNN-based point cloud refinement.

7

Integrated Camera-Conditioned Pipeline

Unified pipeline with multiview diffusion adaptation and depth-based initialization.

8

Feedforward Editing Pipeline

Integration of Qwen2509 + Hunyuan3D for direct text-guided 3D asset generation and editing.
