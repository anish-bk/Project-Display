Frequently Asked Questions

Single Image 3D Reconstruction is a computer vision technique that generates detailed three-dimensional models from a single two-dimensional photograph. Our project uses advanced deep learning methods including diffusion models, Gaussian splatting, and feedforward generation to achieve this.

Our feedforward pipeline can generate 3D models in approximately 25 seconds. The optimization-based pipeline takes longer (45+ minutes) but produces higher quality results with more geometric detail.

For optimal performance, we recommend an NVIDIA A100 GPU with 40GB VRAM. However, the system can run on consumer GPUs like RTX 3090/4090 with reduced batch sizes. Future work includes model compression for broader accessibility.

The project is currently in the final development phase. A demo will be available during the project presentation. For research collaboration inquiries, please use the contact form above.

The system supports multiple output formats including OBJ (with materials and textures), GLB/GLTF (web-ready), PLY (point clouds), and native Gaussian splatting format for real-time rendering.
