Reconstruction

Reconstruction Pipeline Features

Point Cloud Generation

Generate dense 3D point clouds from single images using importance sampling that concentrates points in geometrically significant regions.

Importance Sampling: Stratified sampling based on depth and surface curvature
KNN Refinement: K-Nearest Neighbors for outlier removal and density optimization
Curvature-Based: Alternative sampling approach for smooth surfaces

Score Distillation Sampling

Optimize 3D representations using gradients from pretrained 2D diffusion models, enabling generation of unseen viewpoints.

Camera Conditioning: Explicit viewpoint awareness to reduce artifacts
Gradient-Based: Efficient optimization through differentiable rendering
Noise Scheduling: Adaptive noise levels for stable convergence

Gaussian Representation

Represent 3D scenes using anisotropic Gaussian ellipsoids with learnable parameters for position, shape, opacity, and color.

Anisotropic Gaussians: 3D ellipsoids with full covariance matrices
Spherical Harmonics: View-dependent color representation
Adaptive Density: Dynamic splitting and pruning of Gaussians

Mesh Extraction & Export

Convert optimized Gaussian representations to standard mesh formats with baked textures for use in external applications.

Marching Cubes: Isosurface extraction from density fields
Texture Baking: UV mapping and texture generation
Multiple Formats: OBJ, GLB, PLY export support
